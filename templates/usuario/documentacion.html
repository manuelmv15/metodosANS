📌 Método 1

Este método consiste en aplicar una técnica iterativa para encontrar raíces de funciones no lineales.


Los métodos abiertos emplean una fórmula para predecir la raíz. Esta fórmula puede desarrollarse como una iteración simple de punto fijo, al arreglar la ecuación f(x) = 0 de tal modo que x esté del lado izquierdo de la ecuación:


x = g(x) (ecuación 1)


Esta transformación se realiza mediante operaciones algebraicas o simplemente sumando x a cada lado de la ecuación original. Por ejemplo:


x 2 - 2 x + 3 = 0


se convierte en


x = ( x 2 - 3 ) 2


La utilidad de la ecuación (6.1) es que proporciona una fórmula para predecir un nuevo valor de x en función del valor anterior de x. De esta manera, dado un valor inicial para la raíz xi, la ecuación (6.1) se utiliza para obtener una nueva aproximación xi+1, expresada por la fórmula iterativa.


x i + 1 = g ( x i )


Para obteber el error aproximado de cada iteración se usara el error normalizado:


E a = | x i + 1 - x i x i + 1 | 100 %


Parámetros:

  • Función f(x)
  • Valor inicial x₀
  • Condición de error

¿Cómo usar el Método de Punto Fijo en esta aplicación?

Este módulo permite encontrar una solución aproximada a ecuaciones del tipo f(x) = 0 usando el método iterativo de punto fijo.

Pasos para usarlo:

  • 1. Ecuación g(x): Ingrese la función g(x) en notación de Python. Ejemplo: exp(-x) o cos(x).
  • 2. x₀: Ingrese el valor inicial de la variable x.
  • 3. Error final: Ingrese el error máximo permitido (en porcentaje) para detener el método. Ejemplo: 0.01 para 0.01%.
  • 4. Calcular: Haga clic en el botón para ejecutar el método.

¿Qué muestra la app?

  • El valor aproximado de la raíz.
  • El error relativo porcentual al finalizar.
  • Una gráfica de la función y el punto final alcanzado.

Notas:

  • Si el método no converge, se mostrará un mensaje de advertencia.
  • El ejercicio solo se hara con 15 iteraciones maximo.
  • La aproximacion se hara a cuatro decimales.

📌 Método 2

Este método utiliza diferencias divididas para interpolación polinómica basada en puntos conocidos.

Las fórmulas de Newton-Cotes son los tipos de integración numérica más comunes. Se basan en la estrategia de reemplazar una función complicada o datos tabulados por un polinomio de aproximación que es fácil de integrar:

Requiere:

  • Puntos (x, y)
  • Orden de interpolación
I = ∫abf(x)dx ≅∫abfn(x)dx

donde fn(x) = un polinomio de la forma:

fn(x) = a0 + a1x + ...an-1xn-1+ anxn

donde n es el grado del polinomio.

La integral también se puede aproximar usando un conjunto de polinomios apli cados por pedazos a la función o datos, sobre segmentos de longitud constante.

La regla del trapecio

f1(x) = f(a)+f(b)-f(a)b-a(x-a)

Mediante despejes y agrupaciones obtenemos la formula general:

I = (b-a)f(a)+f(b)2

La regla del trapecio de aplicación múltiple

Una forma de mejorar la precisión de la regla del trapecio consiste en dividir el interva lo de integración de a a b en varios segmentos, y aplicar el método a cada uno de ellos.

h=b-an

Si a y b se designan como x0 y xn , respectivamente, la integral completa se representará como

I = ∫x0x1f(x)dx+∫x1x2f(x)dx+...+∫xn-1xnf(x)dx

Sustituyendo la regla del trapecio en cada integral se obtiene

I = hf(x0)+f(x1)2+hf(x1)+f(x2)2+...+hf(xn-1)+f(xn)2

agrupando tenemos:

I = h2[f(x0)+2∑i=1n-1f(xi)+f(xn)]
