🐍 Python

Python es un lenguaje de programación interpretado, de alto nivel y propósito general. Es ampliamente utilizado en análisis numérico, inteligencia artificial, desarrollo web, y más.

Instalación

# En Windows:
1. Descargar desde https://python.org

📗 NumPy

NumPy es una biblioteca fundamental para cálculos numéricos en Python. Proporciona arrays multidimensionales y funciones matemáticas rápidas.

Instalación

pip install numpy

📊 Matplotlib

Matplotlib es una biblioteca de gráficos en Python para generar gráficos estáticos, animados e interactivos.

Instalación

pip install matplotlib

Ejemplo

import matplotlib.pyplot as plt

x = [1, 2, 3]
y = [2, 4, 6]

plt.plot(x, y)
plt.show()

🧮 Sympy

Sympy es una biblioteca para cálculo simbólico en Python, ideal para álgebra, derivadas, integrales, y más.

Ejemplo

from sympy import symbols, diff

x = symbols('x')
expr = x**2 + 3*x + 2
diff(expr, x)

📌 Método 1

Este método consiste en aplicar una técnica iterativa para encontrar raíces de funciones no lineales.


Los métodos abiertos emplean una fórmula para predecir la raíz. Esta fórmula puede desarrollarse como una iteración simple de punto fijo, al arreglar la ecuación f(x) = 0 de tal modo que x esté del lado izquierdo de la ecuación:


x = g(x) (ecuación 1)


Esta transformación se realiza mediante operaciones algebraicas o simplemente sumando x a cada lado de la ecuación original. Por ejemplo:


x 2 - 2 x + 3 = 0


se convierte en


x = ( x 2 - 3 ) 2


La utilidad de la ecuación (6.1) es que proporciona una fórmula para predecir un nuevo valor de x en función del valor anterior de x. De esta manera, dado un valor inicial para la raíz xi, la ecuación (6.1) se utiliza para obtener una nueva aproximación xi+1, expresada por la fórmula iterativa.


x i + 1 = g ( x i )


Para obteber el error aproximado de cada iteración se usara el error normalizado:


E a = | x i + 1 - x i x i + 1 | 100 %


Parámetros:

  • Función f(x)
  • Valor inicial x₀
  • Condición de error
